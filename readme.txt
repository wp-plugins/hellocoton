== Description ==
Ajoute un lien pour voter pour l'article.
Si votre visiteur possède un compte [hellocoton](http://www.hellocoton.fr), cela aura pour effet d'ajouter votre article sur sa page de recommandation sur [hellocoton](http://www.hellocoton.fr) et d'en informer ses ami(e)s
Dans le cas contraire, le vote est tout de même comptabilisé sur notre plateforme.
